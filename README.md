# MyGameServer(.NetCore)
- 게임서버_C#(.NetCore)
- C#으로 구현한 분산 게임서버

1. 장르: MOBA 를 염두해 두고 구현 진행 중
2. 언어: C# (.NET Core 6.0)
3. 도구: Visual Studio 2022 사용
4. 서버구조: 서버에 대한 기능적 분산 및 중계서버로 구성된 분산 네트워크 게임서버
5. 기타
   * log4net 을 통한 log 저장
   * 패킷 직렬화에 protobuf 사용
   * ObjectPool 사용  
   * 디자인 패턴 (싱글톤, 팩토리 등...)
